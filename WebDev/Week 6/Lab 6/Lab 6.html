Create a Web Page that describes the ways, and implements the ways, that a developer can make a Web site as accessible as possible.

POUR

Perceivable

This content is meant to ensure it is possible that all viewers are able to view and hear any information on the page.

Operable

Operable content means the user is able to interact with it using whatever input device they may have.

Understandable

An Understandable design keeps it consistent and predictable. This way the user isn't struggling to comprehend the content on the page.

Robust

A Robust site allows for multiple accessibility features making it easier for any and all users to use and navigate your a website no matter where or how it is accessed.
